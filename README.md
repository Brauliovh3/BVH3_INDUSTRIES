# 🔮 BVH3 INDUSTRIES — Cyberpunk Portfolio

Portafolio web futurista (estilo cyberpunk) con navegación por secciones, partículas, terminal y un apartado de **Aplicaciones** con descarga de APK.

## ✨ Características

- UI cyberpunk (cyan + magenta) y efectos glow/glitch
- Secciones completas: Inicio, Perfil, Proyectos, Aplicaciones, Habilidades, Contacto
- Partículas en canvas + cursor effect (respeta `prefers-reduced-motion`)
- Descarga de APK desde la sección **Aplicaciones**
